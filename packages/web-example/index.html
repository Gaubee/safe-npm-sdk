npm Registry SDK — safe GET playground

Browsers can't reach registry.npmjs.org directly (no CORS), so this dev server proxies requests via /api → the npm registry, forwarding your token server-side. Only read-only (GET) endpoints are exposed — the proxy rejects every non-GET request, so nothing here can publish, create, or delete.

Credentials

Create a granular read-only token at npmjs.com → Access Tokens. The token and your last operation are saved in this browser's localStorage for convenience; OTP is never saved. Use Clear to wipe both the form and the storage.
